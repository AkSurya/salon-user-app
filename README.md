The Salon Owner App is a dedicated platform designed specifically for salon owners to manage their business within the Salon Booking ecosystem.

This app allows salons to register, manage their profile, handle bookings, and interact with customers who use the User App. It acts as the operational control center for salon partners.

🎯 Purpose

The Salon Owner App enables salons to:

Register and create their business account
List their salon on the User App
Manage services and pricing
Receive and respond to booking requests
View appointments and schedules
Track business performance through a dashboard
🔐 Authentication

Salon owners can:

Sign up using email/phone authentication
Log in securely
Manage their account details
Reset password or verify account
📊 Dashboard

The dashboard provides an overview of:

Total bookings
Upcoming appointments
Completed services
Cancelled bookings
Revenue insights (optional feature)
📅 Booking Management

Salon owners can:

Receive real-time booking notifications
Accept or reject booking requests
View customer details
Manage daily schedule
Update booking status (approved, completed, cancelled)
🛠 Service Management

Salon owners can:

Add new services
Edit service details
Set pricing
Upload service images
Remove unavailable services
🏪 Salon Profile Management

Each salon can:

Update salon information
Add address and contact details
Upload salon images
Set working hours
Manage availability
🔔 Notifications

Salon owners receive notifications when:

A new booking is made
A booking is cancelled
A review is added
🔗 Integration

This app is connected to:

User App – where customers discover and book salons
Admin App – where platform administrators monitor and manage system activity
All apps are connected through a shared backend infrastructure.

🧱 Tech Stack

Flutter (Frontend)
Firebase / Backend (Authentication, Database, Notifications)
GitHub (Version Control)
🚀 Project Vision

The Salon Owner App is built to:

Empower salon businesses
Simplify booking management
Provide seamless communication between customers and salons
Digitize traditional salon operations
